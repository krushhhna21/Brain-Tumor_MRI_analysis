🚀 Learn how I built a Brain Tumor MRI Classifier using Python, TensorFlow & Deep Learning. Detect Glioma, Meningioma, Pituitary, or No Tumor from MRI images with AI + GUI. 🧠

💡 Ever wondered if Artificial Intelligence can help doctors detect brain tumors faster?
In this video, I’ll show you how I built a Brain Tumor MRI Classifier using Python, TensorFlow, and Deep Learning — a complete end-to-end AI project for healthcare. 🧠🩻

We’ll go step by step:
✅ Loading and preprocessing MRI brain images
✅ Training a Deep Learning model with EfficientNetV2B0
✅ Building a classifier for Glioma, Meningioma, Pituitary, or No Tumor
✅ Creating a GUI with Tkinter for easy image upload & prediction
✅ Displaying results with class name and confidence percentage 🎯

This is a full End-to-End Machine Learning Project — perfect for students, beginners, or anyone who wants to add an AI + Healthcare + Computer Vision project to their portfolio.

✨ By the end, you’ll learn how to:
• Prepare and organize a medical imaging dataset
• Train & test a Deep Learning classifier
• Build a user-friendly GUI for predictions
• Deploy AI for practical use in healthcare

📌 Technologies Used: Python, TensorFlow, Keras, OpenCV, Tkinter, NumPy
